# Проект телеграм бота "Решение проблемы глобального потепления"
## Команды бота:
/start - приветствие бота

/help - помощь

/globalhelper - случайное решение проблемы глобального потепления

/fact - случайный факт о глобальном потеплении

/progressfact - случайное достижение в решении этой проблемы

/rules - правила

# Проект телеграм бота "Решение проблемы глобального потепления"
## Команды бота:
/start - приветствие бота

/help - помощь

/globalhelper - случайное решение проблемы глобального потепления

/fact - случайный факт о глобальном потеплении

/progressfact - случайное достижение в решении этой проблемы

/exersize - случайная привычка, помогающая решить проблему

/test - мини тест-игра

## Запуск проекта
### 1) Установка виртуального окружения
```
py -m venv venv
```
```
Set-ExecutionPolicy RemoteSigned -Scope Process
```
```
.\venv\Scripts\Activate.ps1
```

### 2) Подключение необходимых зависимостей
```
pip install -r requiremets.txt
```
### 3) Установка токена

В config.py необходимо вставить токен из @BotFather

### 4) Запуск проекта
```
py main.py
```
